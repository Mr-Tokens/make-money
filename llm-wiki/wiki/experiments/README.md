# Experiments

Experiment pages record concrete attempts.

Each experiment should include:

- Hypothesis
- Date started
- Budget
- Required accounts or wallets
- Steps taken
- Results
- Lessons learned
- Related strategy
- Related raw sources

The default budget is zero until the project has generated proceeds or the user explicitly changes the constraint.
