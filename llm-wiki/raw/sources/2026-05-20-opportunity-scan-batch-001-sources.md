## Scope

This source bundle captures the first batch of opportunities reviewed for the Make Money project. The goal was not to choose a final submission target immediately, but to identify credible zero-capital paths that can create public proof and possible external income.

Project constraints applied:

- No user-provided capital.
- No agent wallet connection, signing, transaction submission, fund movement, private keys, or seed phrases.
- Prefer deliverables that create public evidence: docs, tutorials, reports, templates, small tools, or merged PRs.
- Human-only account registration, grant submission, and payout setup must be separated from agent work.
