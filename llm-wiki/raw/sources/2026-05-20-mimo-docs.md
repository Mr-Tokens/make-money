## Source Note

Xiaomi Mimo is available as an OpenAI-compatible worker provider for this project.

## Public Configuration

- Default base URL: `https://api.xiaomimimo.com/v1`
- Token Plan base URL: read from the user's Mimo subscription page and store in `MIMO_BASE_URL`
- Model: `mimo-v2.5-pro`

## Secret Handling

API credentials are not recorded in the LLM-Wiki. They belong only in ignored local `.env` files when explicitly provided by the user.
