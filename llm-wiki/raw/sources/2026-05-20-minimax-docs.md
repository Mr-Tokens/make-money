# MiniMax Model Docs

URL: https://platform.minimaxi.com/docs/guides/models-intro  
Date accessed: 2026-05-20

## Source Note

MiniMax is available as an OpenAI-compatible worker provider for this project.

## Public Configuration

- Base URL: `https://api.minimaxi.com/v1`
- Primary model: `MiniMax-M2.7`

## Project Policy Update

As of 2026-05-20, active Make Money workflows use only the non-high-speed MiniMax 2.7 model: `MiniMax-M2.7`.

The high-speed MiniMax variant is not an active provider for this project.

## Secret Handling

API credentials are not recorded in the LLM-Wiki. They belong only in ignored local `.env` files when explicitly provided by the user.
