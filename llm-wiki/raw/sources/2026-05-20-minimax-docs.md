## Source Note

MiniMax is available as an OpenAI-compatible worker provider for this project.

## Public Configuration

- Base URL: `https://api.minimaxi.com/v1`
- Primary model: `MiniMax-M2.7`
- Fast model: `MiniMax-M2.7-highspeed`

## Secret Handling

API credentials are not recorded in the LLM-Wiki. They belong only in ignored local `.env` files when explicitly provided by the user.
