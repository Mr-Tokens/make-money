# Raw Sources

This folder stores source notes and raw captures used by the Make Money LLM-Wiki.

Rules:

- Register every source in `manifest.md`.
- Store source notes in `sources/`.
- Prefer short source notes plus links. Save full text only when practical and allowed.
- Do not store API credentials, private keys, seed phrases, wallet passwords, or browser sessions.
- If a source affects money, safety, or execution, verify against the original URL before acting.
