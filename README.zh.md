芝麻是词元君（Mr.Tokens）养的猫，戴着宇航员头盔上班。现在它接到了一项认真任务：从零本金开始，靠可验证的交付，努力买到更好的猫粮。

> 芝麻并不能保证自己能挣到钱，也不能保证大家能跟着我一起挣到钱。

Make Money 的开局很简单：没有本金，不讲暴富，只用真实交付和公开证据往前走。芝麻要做的事也很简单：找机会、做东西、提交结果、记录证据，认真攒出第一笔猫粮钱。

目标很直接：从 Web3 和互联网机会中，找到真实、可交付、可验证的收入路径。项目不靠口号推进，靠机会扫描、交付记录、复盘和证据推进。

## 收入路径 🧭

优先寻找不需要本金、但需要真实产出的机会：

- 公开赏金
- 资助计划
- 社区贡献
- 研究报告
- 小工具
- 文档和教程
- 服务线索

每个机会都要先回答三个问题：

1. 是否真实可靠。
2. 是否能在零本金条件下交付。
3. 是否能留下证据、作品或下一步线索。

## 工作方式 ⚙️

团队执行流程：

1. 扫描机会。
2. 保存来源。
3. 评分排序。
4. 选择最小可交付成果。
5. 在合适时把研究、整理、文案、总结或风险检查分配给 Mimo 和 MiniMax。
6. 由 Codex 审核 worker 输出，再写入长期项目记录。
7. 提交或沉淀为项目资产。
8. 写入 LLM-Wiki。

README 只展示简版进展；完整过程进入 LLM-Wiki。

## 判断标准 📌

项目是否有进展，看三件事：

- 是否发现了更好的机会。
- 是否完成了真实提交。
- 是否出现了收入证据或明确付费线索。

收入只按证据记录：确认、到账、或明确的付费机会。没有证据时，状态仍然记为 0。
