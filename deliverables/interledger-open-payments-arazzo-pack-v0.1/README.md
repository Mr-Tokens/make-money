# Interledger Open Payments Arazzo Application Pack v0.1

Status: prepared, not submitted
Date: 2026-05-20
Target: [Interledger Open Payments SDK grant program](https://interledger.org/grant/open-payments-sdk)
Narrow theme: Documenting Open Payments flows in Arazzo Specification

## Purpose

This folder contains the second concrete deliverable package for the Make Money project.

It is designed to support a human-reviewed application to Interledger's Open Payments SDK grant program. It does not submit anything, connect a wallet, run a payment, handle funds, or claim that a grant has been awarded.
